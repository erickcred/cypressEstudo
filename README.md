<h1>Instalação do Cyress</h1>

<p>Processo para instalação do Cypress</p>
<pre>
  <code>
    npm init -y
    npm install cypress --save-dev
  </code>
</pre>

<p>Rodando o Cypress</p>
<pre>
  Vamos configurar no arquivo (package.json) e depois no terminal rodamos npm run open
  <code>
    "open": "cypress open",
  </code>
</pre>
